This workflow starts a local HTTP server to host the portal files.

### 1. Identify Project Root
Ensure you are in: `C:\Users\Ale\Dropbox\clases_universidad\cuantum32\Quantum32 Hybrid Sampler\`

### 2. Run Server
// turbo
Run the following command to start the server on port 8000:

```powershell
python -m http.server 8000
```

### 3. Access the Portal
Open your browser and go to: [http://localhost:8000](http://localhost:8000)
